## We're adapting International Women's Day Bingo to help folks celebrate [International Women's Day](https://www.internationalwomensday.com/)!
    

## Running
### Prerequisites
 - npm
 - Java 8
 
#### UI
 1. `cd ui`
 1. `npm install`
 1. `npm run start`
 1. Navigate to `http://localhost:8000`
#### Backend
 1. `cd backend`
 1. `./gradlew bootRun`
